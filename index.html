Meu Aprendizado

Um dos primeiros aprendizado que tive e continuo aprendendo é comunicar através de vídeo, já na entrevista de seleção somos incentivados a utilizar o video, isso está sendo muito importante no meu processo de crescimento pessoal.

Os momentos de soft skills nos motivam a desenvolver outras habilidades além do desenvolvimento de códigos. Já vimos a importância do autoconhecimento e da empatia para o bom desempenho nas atividades em equipe. Habilidade muito requisitada no mundo do trabalho.

Sobre conhecimentos técnicos eu já tinha uma boa base, mesmo assim consegui agregar muito mais. Inclusive precisava automatizar um processo usando shell script, há muito tempo estava pesquisando sem sucesso, coloquei minha dificuldade no slack e prontamente o colega Felipi Firmino me ajudou.

O GIT e GITHUB era algo que conhecia muito pouco. Na segunda semana do curso já estava conseguindo entender bem e subir os codigos para repositórios.

Não tinha muita noção sobre versionamento, e as aulas me ajudaram a compreender a importancia de utilizar o GIT para criar branch, commit e push.

O mais desafiador foi o GITHUB, já tinha uma conta na plataforma mas não tinha utilizado para os fins de repositório. Na verdade nunca tinha aprofundado nesse assunto. Nas aulas ficou claro a importância de criar repositórios e compartilhar os codigos com equipe e colegas.

Quanto a HTML e CSS apesar do meu bom conhecimento, consegui aprender tecnicas importantes que desconhecia.

No CSS não conhecia a propriedade "display:inline-block" que posibilita elementos inline receber comportamentos de elementos block.

O HTML semântico não tinha muito aprofundamento, consegui entender a importância de utilizar as tags corretas para a melhor localização e entendimento dos motores de busca.

O mais legal é sabar que estamos utilizando as mesmas técnicas e ferramentas utilizadas por empresas de desenvolvimento web no mundo.
